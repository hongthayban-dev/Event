Do not run `git add`, `git commit`, or `git push` automatically after making code changes.

**Why:** User prefers to push to GitHub themselves.

**How to apply:** Make the code changes, then stop. Do not offer to push or ask about pushing. Only commit/push if the user explicitly asks ("push ขึ้น github", "commit", "push ให้หน่อย", etc.).
